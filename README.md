# 40code_Eureka_ext
[Eureka](https://github.com/EurekaScratch/eureka-loader/blob/trunk/README-zh_CN.md) 是一个全平台的 Scratch 扩展加载器。该项目计划用来提供40code **编辑器内置拓展** 的等价 Eureka 版拓展。  

相信这个有趣的项目可以给 40code 的创作者带来更多的乐趣。
